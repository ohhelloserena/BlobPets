# Blob Pets

An Android mobile virtual pet game where players can raise blobs and fight other players' blobs.

[![download](https://play.google.com/intl/en_us/badges/images/generic/en_badge_web_generic.png)](https://play.google.com/store/apps/details?id=com.Team2.BlobPets&hl=en&pcampaignid=MKT-Other-global-all-co-prtnr-py-PartBadge-Mar2515-1)

## Requirements

* Unity
* Android SDK
* Laravel
